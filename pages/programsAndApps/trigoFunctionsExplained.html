TRIGONOMETRIC FUNCTIONS VISUALIZED

An educational web-app aiming to help you grasp the concepts of trigonometric functions

View in Browser

Technical Summary

  • Type: Web Application
  • Languages: JavaScript, p5.js
  • Link: PROJECT LINK

Overview

Introduction

I got into programming way back in highschool and one of my first project that gained praise was this interactive simulation I created to visualize trigonometric functions. I created it on KhanAcademy where they allow new learners to show their programming skills by creating visualization projects. This project is one of those. :)

How to use

You may choose the units of the angles displayed as well as it's direction/sign on the left part. On the right side, you can choose which trigonometric function to show. You can also see some display options such as toggling gridlines and labels. Below that is a Reset button in case you want to reset the view.

On the bottom, you can see a cartesian graph with angles on the x-axis and the value on the y-axis. It will show the graph of the trigonometric function as well as the point where the selected angle is at on the graph.
