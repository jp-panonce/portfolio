Guess the RGB

An entry for the "One HTML Page Challenge", this game tests your ability to identify the RGB of a color.

Play in Browser

Technical Summary

  • Type: Web Application, Game
  • Languages: HTML, CSS, JavaScript
  • Link: https://jp-panonce.github.io/OnePageHTMLs/guessTheRGB.html

Overview

Introduction

This game is my first entry for the One HTML Page Challenge which is a challenge for Front-end web developers to create something interesting with only one HTML page and with a set maximum HTML file size.

It is a guessing game to test the player's ability to guess the RGB values of a color (set as the background color of the page).

How to play the game

To play the game, visit this URL. Once in, a new game starts. The player has to guess the RGB color of the site's background color by entering the guessed R,G,B values to each respective input fields. To submit the guess, the player would have to press on the 'Enter' key on their keyboard. The score updates based on correctness of the guess and immediately starts the next round.

Each new game consists of 10 rounds of guessing after which, will display the player's total score and a button to start a new game. The score per round is computed as follows:

Implementation

HTML & CSS

The game has a simple and straightforward interface. The game page shows the title, current score and rounds played. Below which has 3 input fields for the player to enter his/her guess.

JavaScript

When the page is refreshed, a new game immediately starts. The R, G and B values of background color is determined randomly using this function: Math.random(0,255). The user then enters his/her answer and upon pressing the "Enter" key, will trigger an EventListener function. Said function gets the RGB values entered by user and compares it to the RGB values of the background color.

The score is then updated by adding the round score determined using the formula already mentioned. A counter variable to check # of rounds played is also updated.

Once a total of 10 rounds has been played, the End Screen will then display which shows the player's total score.
